Attendance Management System

"This project, developed in collaboration with Sean Archer and my other groupmates using the Python programming language, focuses on creating an Attendance Management System for Barangay Martirez Del 96 in Pateros, Metro Manila, Philippines. The system comprises two subsystems: the Admin System and the Face Recognition System. Administrators can create and register users, monitor attendance, and manage logins/logouts through the Admin System. Simultaneously, users can seamlessly log in and out using facial recognition technology in the Face Recognition System. I acknowledge the valuable contributions of my other groupmates to the successful development of this system."
